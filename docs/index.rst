LineRate
========

LineRate is a Python package for computing thermal rating of overhead conductors. It contains an
implementation of the heat balance equations in CIGRE techincal brochure 601 :cite:p:`cigre601`.
Implementations of other models, such as the IEEE Std 738-2012 :cite:p:`ieee738` may come at a
later time.
